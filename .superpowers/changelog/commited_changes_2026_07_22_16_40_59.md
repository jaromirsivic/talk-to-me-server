# Necommitované změny

- Přidán endpoint `POST /api/v1/queueInfo` s režimy `min` a `max` a konzistentním snímkem aktivní fronty.
- Do webového portálu přidáno lokalizované tlačítko Stop, sekundové dotazování stavu fronty a ochrana proti souběžným požadavkům.
- Doplněny styly, cachebuster webových prostředků a překlady tlačítka ve všech podporovaných jazycích.
- Rozšířeny testy doménové vrstvy, API a webového portálu a aktualizována lokální API dokumentace.
